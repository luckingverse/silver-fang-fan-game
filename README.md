# Hokkyoku: Star Fang

A browser platformer fan game based on *Ginga: Nagareboshi Gin* (Silver Fang).

Growing up, *Ginga: Nagareboshi Gin* was my absolute favorite anime — the epic story of Gin and his pack fighting for survival and honor left a huge mark on me. As a kid, I always wished for an interactive game to live out those battles and adventures. Years later, I finally built this as a love letter to the series: a simple browser-based platformer where you can run, jump, bite, and unleash special moves across 10 worlds with boss fights, a skins system, and even a leaderboard.

▶ **Play Online:** https://luckingverse.github.io/silver-fang-fan-game/

## Controls
- **WASD / Arrow keys** — Move
- **Space / W** — Jump (supports double/triple jumps)
- **X** — Bite
- **Z** — Battouga
- **C** — Army (summon/call allies?)
- **Down + in air** — Ground Pound
- **B** — Vending Machine (item/shop?)

Built with HTML5/JavaScript (with assistance from Claude AI). Feedback welcome — pull requests or issues appreciated!
